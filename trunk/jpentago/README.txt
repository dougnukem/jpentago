This is JPentago source distribution.


Prerequisites:
- You need Apache ANT and JDK 5 installed on your system and configured on your PATH.

Building:
To compile run:
ant

Running:
To run the 3D view of JPentago, after building the source code you can run
jpentago.bat (for windows)
jpentago.sh  (for Linux/Mac OSX)

If for some reason those don't work


Setting up in Eclipse IDE:
If you retrieved this source code from the subversion repository, then it will have contained
the Eclipse project, which you can import into your Ecliipse workspace, File >> Import >> Existing Project into Workspace.
